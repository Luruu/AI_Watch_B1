 \doxysection*{AI Watch B1 }

   \doxysubsubsection*{AI Watch B1 is a module of the Ai\+Watch project that includes the implementation of the digital twin and software for detecting outliers on an infinite data stream. Eclipse Ditto receives the coordinates from the Ai Watch A1 module, updates the digital twin and send them to the Streaming Anomaly Detector. This software sends the coordinates with the \char`\"{}anomaly\+\_\+value\char`\"{} flag to the Ai Watch C1 (Unity 3D) module. }
